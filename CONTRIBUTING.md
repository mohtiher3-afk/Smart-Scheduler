# Contributing to Smart Scheduler

We welcome contributions to Smart Scheduler! Please follow these guidelines to ensure a smooth development process.

## How to Contribute
1. **Report Issues**: Use the issue tracker for bug reports and feature requests.
2. **Propose Changes**: Open a pull request with a clear description of the changes.
3. **Follow Style Guides**: Ensure code adheres to `CODE_STYLE.md`.
4. **Test**: Run all tests before submitting.
